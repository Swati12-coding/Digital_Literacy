# 📚 Digital Literacy Project

## 📌 Project Overview
This project focuses on understanding and applying digital literacy concepts. It covers topics such as safe internet practices, professional communication, cybercrime awareness, and the use of digital tools. The project includes practical tasks like creating an infographic, quiz, emails, case study, and prevention checklist.

---

## 🧩 Modules Covered
- Digital Literacy Awareness (Infographic using Canva)
- Email Writing and Etiquette
- Social Media Do’s and Don’ts
- Cybercrime Case Study (UPI/Online Payment Fraud)
- Prevention Checklist for Online Safety
- Quiz Creation (Google Forms)
- Coding Practice (HackerRank)
